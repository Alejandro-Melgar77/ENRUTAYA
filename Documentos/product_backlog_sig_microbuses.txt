PRODUCT BACKLOG – SIG MICROBUSES (SPRINT ÚNICO MVP)

OBJETIVO DEL SPRINT:
Desarrollar una aplicación funcional que permita visualizar rutas de microbuses, ubicación en tiempo real de unidades y cálculo básico de ETA.

--------------------------------------------------
SECCIÓN 1: CONFIGURACIÓN INICIAL

PB01 - Configuración del proyecto Flutter
- Crear proyecto Flutter
- Configurar estructura de carpetas (features, services, models)
- Configurar variables de entorno

PB02 - Integración con Supabase
- Crear proyecto en Supabase
- Configurar conexión desde Flutter
- Instalar SDK de Supabase en Flutter

PB03 - Configuración de Mapbox
- Crear cuenta en Mapbox
- Obtener API Key
- Integrar SDK en Flutter
- Mostrar mapa base

--------------------------------------------------
SECCIÓN 2: BASE DE DATOS

PB04 - Creación de tablas
- Crear tabla lineas
- Crear tabla unidades
- Crear tabla posiciones_actuales

PB05 - Inserción de datos (Seeder)
- Insertar líneas con rutas (GeoJSON)
- Insertar unidades con códigos únicos

PB06 - Configuración PostGIS
- Habilitar extensión PostGIS
- Validar almacenamiento de geometrías

--------------------------------------------------

- Validar campos
- Conectar con base de datos

PB08 - Validación de unidad activa

- Bloquear acceso si ya está activa

PB09 - Selección de sentido
- UI para elegir ida/vuelta
- Guardar selección

PB10 - Inicio de recorrido
- Insertar registro en posiciones_actuales
- Activar estado de transmisión

PB11 - Envío de GPS
- Obtener ubicación con GPS del dispositivo
- Enviar latitud, longitud, velocidad cada 30s
- Actualizar registro en BD

PB12 - Finalizar recorrido
- Eliminar registro de posiciones_actuales

PB13 - Marcar desvío
- Actualizar estado a 'desvio'

PB17 - Ubicación del usuario

- Mostrar en mapa

PB18 - Cálculo punto más cercano

PB19 - Cálculo de ETA
